Getting Started:

	1. Clone this repository into ~/catkin_ws/src/
		a) cd ~/catkin_ws/src
		b) git clone https://github.com/NasaSpaceGrantRobotics/MATE19.git
	2. Make the Packages
		a) cd ~catkin_ws
		b) catkin_make
	3. Reset ROS Environment Variables
		a) source ~/catkin_ws/devel/setup.bash
	4. Check that the Packages are installed
		a) rospack find driverside
		b) rospack find robotside
